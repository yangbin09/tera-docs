## 📜 原文
```
醉里挑灯看剑
梦回吹角连营
八百里分麾下炙
五十弦翻塞外声
沙场秋点兵
马作的卢飞快
弓如霹雳弦惊
了却君王天下事
赢得生前身后名
可怜白发生
```
## 🎯 主题
《破阵子 · 为陈同甫赋壮词以寄之》的主要主题思想是表达作者辛弃疾收复失地、报效国家的壮志豪情。词中通过描绘军旅生活的壮阔场景和驰骋沙场的英雄气概，展现了词人渴望为国建功立业的强烈愿望。然而，结尾"可怜白发生"一句又透露出壮志未酬的悲愤与无奈，反映了南宋爱国志士面对国事日非的苦闷心情。这首词既是作者个人情怀的抒发，也是对友人陈亮的勉励，体现了共同的爱国志向和英雄气概。
## 🏗️ 结构
《破阵子·为陈同甫赋壮词以寄之》在结构上采用了典型的双调形式，上下片各五句，共十句，整体布局严谨而富有变化。

上片以军旅生活场景开篇，首句"醉里挑灯看剑，梦回吹角连营"通过"醉里"和"梦回"两个时间点，营造出词人即使在醉梦中也不忘军旅生活的氛围。接着"八百里分麾下炙，五十弦翻塞外声"进一步展开军营生活场景，描绘了将士们的生活和军乐声。最后以"沙场秋点兵"作结，点明上片的核心内容——军旅生活和战争准备，形成了一个完整的叙事层次。

下片则转向抒发壮志豪情和人生感慨。"马作的卢飞快，弓如霹雳弦惊"两句描绘战场上的激烈战斗场景，用动态的描写展现了战争的紧张与激烈。"了却君王天下事，赢得生前身后名"直抒胸臆，表达了词人渴望为国家建功立业的雄心壮志。最后以"可怜白发生"突然转折，表达了壮志未酬、年华已逝的悲凉感慨，与上片的豪迈形成鲜明对比，形成强烈的情感冲击。

全词在结构上呈现出明显的时空转换：从"醉里"到"梦回"，再到军营、战场，最后回到现实中的"白发生"，实现了时间和空间的自然流动。同时，词中虚实结合，上片多为实写军旅生活场景，下片既有实写的战场场景，又有虚写的理想抱负，最后以现实感慨作结，虚实相生，层次分明。

节奏变化也是这首词的重要结构特点。上片节奏较为舒缓，通过场景的铺陈营造出军旅生活的氛围；下片节奏加快，尤其是"马作的卢飞快，弓如霹雳弦惊"两句，用短促有力的语言描绘了激烈的战斗场景；最后以"可怜白发生"的沉郁感慨作结，节奏再次放缓，形成起伏变化，增强了词的艺术感染力。

整体而言，这首词结构严谨，层次分明，从豪迈壮阔的军旅生活到激烈的战场描写，再到建功立业的雄心壮志，最后突然转折到悲凉感慨，形成了强烈的情感对比，充分体现了辛弃疾词作的豪放风格和艺术特色。
## ✍️ 语言风格
《破阵子·为陈同甫赋壮词以寄之》是辛弃疾豪放词风的代表作，其语言风格特点鲜明而独特。

用词方面，此词以刚健有力为主要特色。词中大量使用军事术语，如"剑"、"营"、"炙"、"弦"、"沙场"、"马"、"弓"等，直接展现了词人的军旅背景和报国情怀。这些军事词汇不仅具体形象，而且充满阳刚之气，构成了词作雄浑豪放的基础。数量词的运用也颇具特色，"八百里"、"五十弦"等词语增强了词作的气势和画面感，使读者仿佛置身于壮阔的战场之中。

修辞手法上，辛弃疾运用了对偶、比喻、夸张等多种修辞技巧。开篇"醉里挑灯看剑，梦回吹角连营"形成工整的对偶，既增强了节奏感，又生动描绘了词人日夜思战的场景。"弓如霹雳弦惊"运用比喻，将弓弦的声音比作霹雳，形象地再现了战场上的紧张激烈气氛。而"了却君王天下事"则运用夸张手法，表现出词人收复失地、报效国家的宏大抱负。

在语言节奏上，整首词气势恢宏，节奏明快。特别是下阕的"马作的卢飞快，弓如霹雳弦惊"，短句连用，一气呵成，生动地描绘了战场上的紧张激烈。而结尾处"可怜白发生"突然放缓节奏，形成强烈对比，突出了理想与现实的落差，给人以深刻的情感冲击。

情感表达上，此词壮怀激烈而又沉郁苍凉。上阕描绘了壮阔的战场场景，表达了词人渴望收复失地、报效国家的壮志豪情；下阕则通过"了却君王天下事，赢得生前身后名"进一步抒发理想，而结尾"可怜白发生"却流露出壮志未酬的悲愤和无奈，形成了理想与现实的鲜明对比。

整体风格上，这首词体现了典型的豪放派特色，气势磅礴，意境开阔。在豪放中带有沉郁，在激烈中带有顿挫，特别是结尾处情感的突然转折，展现了辛弃疾词作沉郁顿挫的艺术魅力。词中军事意象的密集运用，不仅展现了词人的军旅生涯，更深刻地表达了他的爱国情怀和报国无门的悲愤。

《破阵子·为陈同甫赋壮词以寄之》的语言艺术，在于它将豪放的气势与深沉的情感完美结合，通过刚健有力的用词、多样的修辞手法和富有变化的节奏，塑造了一个壮怀激烈的爱国词人形象，也展现了南宋时期有志之士收复失地的共同愿望和无奈现实。
## 📚 文化札记
《破阵子》是词牌名，源自唐代教坊曲，本为舞曲，后用作词调。题目"为陈同甫赋壮词以寄之"中的陈同甫即陈亮（1143-1194），字同甫，号龙川，南宋著名思想家、文学家，与辛弃疾为志同道合的好友，都力主抗金收复失地。"壮词"指豪迈雄壮的词作，表明这首词是辛弃疾为好友陈亮而作，寄托了共同的抗金理想。

"醉里挑灯看剑"中的"挑灯"是点亮油灯的动作，古人多用油灯照明；"看剑"象征渴望上阵杀敌，收复失地。辛弃疾作为爱国将领，时刻不忘抗金复国大业。

"梦回吹角连营"中的"角"是古代军中的号角，用于军事活动和鼓舞士气；"连营"指军营相连，形容军队规模宏大。此句描写了辛弃疾梦中回到军营，听到号角声的场景，表达了对军旅生涯的深切怀念。

"八百里分麾下炙"运用了典故，"八百里"出自《晋书》中王济以"八百里駃"（骏马名）作为赌注的典故，这里指代牛，表示丰盛的军中伙食；"麾下"指部下、将帅的旗帜之下；"炙"即烤肉。此句描绘了军中将帅与部下共享烤肉的豪迈场景。

"五十弦翻塞外声"中的"五十弦"指瑟，古代的一种弦乐器，有五十根弦，这里泛指军中乐器；"塞外声"指边塞地区的音乐，多描写战争和边塞生活，风格雄壮悲凉。此句描写军中演奏边塞乐曲的雄壮场面。

"沙场秋点兵"中的"沙场"是战场，古时多指沙漠或平坦开阔的战场；"点兵"是检阅军队，分配任务。古代秋季多进行军事活动，称为"秋点兵"，此句点明了季节和军事背景。

"马作的卢飞快"中的"的卢"是名马，相传三国时刘备所骑的马，曾"的卢跃檀溪"，救刘备于危难。此处用典，表达了渴望拥有良马驰骋沙场的愿望。

"弓如霹雳弦惊"中的"霹雳"是雷声，形容弓弦拉开时发出的声音如同雷鸣，形容弓力强劲；"弦惊"指拉开弓弦时发出的声音。此句生动描写了射箭时的壮观场景。

"了却君王天下事"中的"君王天下事"指辅佐君王完成统一天下的大业，这里特指收复失地、恢复中原的抗金事业。此句表达了辛弃疾渴望为国家建功立业的雄心壮志。

"赢得生前身后名"指在生前和死后都能获得不朽的声名，这是古代士人追求的人生价值之一。此句表达了辛弃疾希望通过抗金建功立名，流芳百世的愿望。

"可怜白发生"中的"可怜"是可惜、可叹之意；"白发"指年老。此句表达了辛弃疾虽然壮志未酬，但已年老，感叹时光流逝，壮志难酬的悲愤。

这首词写于南宋时期，当时北方金朝占领了中原大片领土，南宋朝廷偏安江南，主和派当政。辛弃疾和陈亮都是主张抗金的爱国志士，但他们的主张不被朝廷采纳。词中气势恢宏，豪放悲壮，是辛弃疾"豪放派"词风的代表作，运用了丰富的军事意象，展现了词人的军旅情怀和爱国精神。
## 🌅 创作背景
《破阵子 · 为陈同甫赋壮词以寄之》创作于南宋孝宗淳熙十五年（1188年），地点在江西铅山。当时辛弃疾与陈亮（字同甫）在此地会面，两人志同道合，共商恢复大计。分手后，辛弃疾写下此词寄给陈亮，以抒发彼此共同的抗金壮志。

辛弃疾当时的生活状况颇为失意。自1181年被罢官后，他已在江西上饶带湖闲居长达七年之久。作为一位力主抗金的爱国志士，他虽有收复中原的雄心壮志，却遭到朝廷主和派的排挤，无法施展抱负。此次与陈亮的会面，使他在孤独的闲居生活中难得地找到了知己，两人纵论国事，慷慨激昂，使辛弃疾压抑多年的爱国热情得以暂时释放。

从历史背景看，南宋朝廷正处于偏安一隅的状态。宋孝宗虽有北伐之心，但符离之战失败后，朝廷逐渐转向保守，主和派占据上风。北方金国虽然强大，但内部矛盾日益加剧，这本是南宋收复失地的良机，然而朝廷却选择妥协退让，不思进取。整个社会风气萎靡，爱国志士虽有报国之心，却无用武之地。

这首词的创作动机源于辛弃疾与陈亮共同的抗金志向和对时局的忧虑。词中"醉里挑灯看剑，梦回吹角连营"等军旅场景的描写，正是对往日军旅生涯的回忆和对未来战斗的憧憬。"了却君王天下事，赢得生前身后名"直接表达了收复失地的强烈愿望，而最后"可怜白发生"则流露出壮志未酬的悲愤与无奈。整首词气势磅礴，豪情万丈，既是对友人的寄怀，也是对国家命运的深切忧虑。

## 🔍 赏析
《破阵子 · 为陈同甫赋壮词以寄之》是辛弃疾豪放词的代表作之一，其意境营造雄浑壮阔，充满英雄气概。上阕"醉里挑灯看剑，梦回吹角连营"以虚实结合的手法，通过"醉"与"梦"两个场景，将词人内心深处的军旅情怀与现实的无奈巧妙融合。"八百里分麾下炙，五十弦翻塞外声。沙场秋点兵"则生动描绘了军营生活的壮阔场景，营造出金戈铁马、气势恢宏的意境，展现出词人对军旅生活的无限向往。

语言特色上，辛弃疾运用了雄浑豪放、气势磅礴的语言风格。"马作的卢飞快，弓如霹雳弦惊"运用比喻和夸张的修辞手法，将战场上的紧张激烈表现得淋漓尽致，形象生动，极具画面感。词中还巧妙运用对比手法，如"了却君王天下事，赢得生前身后名"与"可怜白发生"形成鲜明对比，突显了理想与现实的巨大差距，增强了词作的悲剧色彩和感染力。

情感表达上，这首词抒发了词人渴望收复失地、报效国家的壮志豪情，同时也流露出壮志未酬的悲愤与无奈。上阕通过军营生活的描写，表现了词人对军旅生活的热爱；下阕通过战场场景的描绘，表达了为国效力的决心；结尾"可怜白发生"则流露出对时光流逝、壮志未酬的感慨，展现了词人深沉的家国情怀和英雄气短的人生悲剧。

在文学价值和影响意义上，这首词以其雄浑的气势、生动的意象、真挚的情感，成为宋代豪放词的典范之作。它不仅影响了宋代以后的词创作，也激励了无数爱国志士。词中表现出的爱国精神和英雄气概，成为中国传统文化中的重要精神财富，对后世文学创作和民族精神塑造都产生了深远影响，彰显了中国古代知识分子的家国情怀和责任担当。
